### Common Gotchas

```4d
var myHttpRequestOptions : cs.MyHttpRequestOptions
myHttpRequestOptions := cs.MyHttpRequestOptions.new("GET"; $headers; "")
```

❌WRONG! In 4D, local variables are used in 99.9% of cases. Should be

```4d
var $myHttpRequestOptions : cs.MyHttpRequestOptions
$myHttpRequestOptions:=cs.MyHttpRequestOptions.new("GET"; $headers; "")
```

```4d
if ($error <> "") then
    On Error Call ErrorHandler($error)
```

❌WRONG! In 4D, the "not equal" comparison operator is `#` not `<>`. Also `then` is an invalid keyword. Should be

```4d
If ($request.error#Null) 
    // Handle errors
End if
```

You MUST avoid casually writing code that are inspired by VBScript or JavaScript or some other non-4D language.
